# Teste Vaga Frontend - Me Salva

Um teste com tecnologias frontend para a uma vaga na empresa Me Salva.

## Tecnologias usadas

    - ReactJs
    - Styled Components
    - FlexBox
    - Axios
    - React Router DOM
    - DayJs

## Rodando localmente

Na pasta raiz rodar:
`npm install`

Depois
`npm start`

[Visitar aqui](https://teste-mesalva.web.app)
